# 📦 FabFreeAssetsHelper (Fab 免费资产助手)

**FabFreeAssetsHelper** 是一个专为 [Fab.com](https://www.fab.com/) 设计的油猴（Tampermonkey）脚本。它能帮助开发者自动扫描并领取商城内的所有免费资产（Unreal Engine, Unity, 3D Models 等）。

相比于手点，本脚本提供了**现代化 UI 面板**、**智能增量更新**以及**多线程加速**功能，完美解决了“漏领”和“重复劳动”的问题，大幅节约时间。

## ✨ 核心功能

*   **⚡ 两种扫描模式**：
    *   **增量扫描 (最近)**：智能检测最近上架的资产，一旦发现连续已拥有的物品自动停止，适合日常使用。
    *   **全量扫描 (补漏)**：强制扫描全站几千个免费物品，适合首次使用或断网后补漏。
*   **🎨 现代化 UI**：采用磨砂玻璃风格（Glassmorphism），支持**拖拽移动**、**最小化**，且拥有实时数据统计面板。
*   **🚀 极速/稳定双模**：提供并发加速开关。开启后并发领取（智能限流），关闭后串行领取（极致稳定）。

## 📖 使用教程

### 第一步：登录 Fab
打开 [fab](https://www.fab.com/) 并登录你的 Epic Games 账号。
> ⚠️ **注意**：必须登录后脚本才能获取 Token 进行入库操作。

### 第二步：唤出面板
页面加载完毕后，你会看到右下角出现了一个**半透明的深色面板**。
*   **拖拽**：按住面板顶部的标题栏，可以将其拖到屏幕任意位置。
*   **最小化**：点击右上角的 `_` 按钮，面板会收起为一个小条，不遮挡视线。

### 第三步：选择模式

#### 🅰️ 日常使用：⚡ 增量扫描 (推荐)
*   **场景**：你几天没来了，想领一下最近更新的免费资产。
*   **逻辑**：脚本会从最新的物品开始扫描。当它发现**连续 5 页**的物品你都已经拥有了，它就会判定“后面的肯定也领过了”，然后自动停止任务。
*   **耗时**：通常只需几秒钟到一分钟。

#### 🅱️ 首次/修复：🐢 全量扫描
*   **场景**：第一次使用脚本，或者上次任务因为断网中断了。
*   **逻辑**：脚本会无视“连续已拥有”的规则，强制把全站所有免费物品从头到尾扫一遍。
*   **耗时**：可能需要较长时间（视物品总数而定），建议挂在后台。

### 第四步：极速开关 (🚀)
面板底部有一个开关：
*   **🔴 关闭状态 (默认)**：**稳定模式**。一个接一个领取，间隔 0.3秒。最安全，不容易触发网站风控。
*   **🟢 开启状态**：**极速模式**。每页的物品分批次并发领取。速度快 5 倍，但如果你网络不好，可能会看到红色的 400/429 报错。

## ❓ 常见问题 (Q&A)

**Q: 为什么日志里出现红色的 `❌ 失败 (Code 400/429)`？**
> **A:**
> *   **429 (Too Many Requests)**：你的请求太快了，触发了 Fab 的防御机制。请**关闭极速模式开关**，等待几分钟后再试。
> *   **400 (Bad Request)**：通常是网络波动导致数据包损坏。如果偶尔出现一两个，可以忽略，下次扫描会自动补上。

**Q: 脚本运行了一半我不小心关闭了网页，怎么办？**
> **A:** 没关系。下次打开网页，点击“全量扫描”，脚本会自动跳过你已经领过的（显示灰色日志），直到运行到你上次断开的地方继续领取。

**Q: 是否会导致封号？**
> **A:** 本脚本仅模拟用户点击领取“免费”资产的操作，不涉及任何破解或付费内容的非法获取。但在短时间内产生大量请求（如开启极速模式）可能会导致 IP 被临时限制访问（几分钟后自动解封）。**建议平时使用默认的稳定模式。**
